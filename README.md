📝 Notes App

A full-stack Notes Application with authentication and CRUD functionality.

---

🚀 Features

- 🔐 User Authentication (JWT)
- 📝 Create, Read, Update, Delete Notes
- 🔍 Search Notes with Highlight
- 🌙 Dark Mode
- 📱 Responsive Design

---

🛠 Tech Stack

Frontend

- HTML
- CSS
- JavaScript

Backend

- Flask
- SQLite
- JWT Authentication
